## üöÄ Featured Projects

### **‚ö° Power & Thermal Management Research**

#### 1. **CPINN-ABPI: Physics-Informed Neural Networks for Accurate Power Estimation**
- First hardware validation on NVIDIA Jetson Xavier AGX
- 84.7% MAE reduction (CPU), 73.9% (GPU)
- Sub-millisecond inference with NSGA-II multi-objective tuning
- Application: Real-time power profiling in multicore MPSoCs
- **Status**: Published research

#### 2. **Cluster-BPI: Fine-Grain Blind Power Identification**
- IEEE HPEC 2024 publication
- 77.56% error reduction in power estimation
- DBSCAN-enhanced framework for thermal Trojan defense
- Hardware security for multicore SoCs
- **Status**: Production-ready framework

#### 3. **Fine-Grained Clustering-Based Power Identification for Multicores**
- 76% error reduction over traditional methods
- DBSCAN clustering + NMF for accurate power profiling
- Steady-state thermal analysis
- **Status**: arXiv preprint (2409.18921)

#### 4. **Thermal Vulnerability of 3D-Stacked High-Bandwidth Memory**
- Analysis of convergent thermal-wave attacks in HBM
- RC lattice thermal modeling
- Memory subsystem security assessment
- **Status**: arXiv preprint (2509.00633)
